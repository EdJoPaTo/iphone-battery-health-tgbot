# iPhone Battery Health Telegram Bot

A small Telegram Bot for people I know to quickly update their own devices regarding Battery Health.

See also the data repository which is updated by this project: <https://github.com/EdJoPaTo/iPhoneBatteryHealth>

The source code here is only meant to be open source. It's not documented or meant to be used otherwise.

A notification feature is not planned.
It would require additional state and writing everyone regularly also has the nice benefit of socializing with them ;)
